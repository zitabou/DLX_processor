# =========================================
# Simulation
#==========================================

1) Put the .asm file in the assembler.bin folder 
2) From the assembler.bin run the assembler.sh indicating the .asm you want to simulate. (The instructions will be put automatically in the iram.mem and used in the simulation)
3) In the DLX directory run resim.bash (this will open the simulation and use correct directories)
4) run simDLX.bash in the questasim terminal

# =========================================
# Synthesis
#==========================================

1) from the DLX directory run resyn.bash (this will open the terminal in the syn directory)
2) run design_vision
3) run synDLX.bash in the terminal
